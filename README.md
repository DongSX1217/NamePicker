## FAQ
Q:没有找到names.txt，请参照README进行处理

A:将names_example.txt复制一份并将复制的那份改名为names.txt，随后按照names.txt中内容操作

Q:配置界面和主界面风格不同

A:作者目前没找到怎么修，但不影响使用

Q:杀毒软件认为这是病毒软件

A:将该软件添加至杀毒软件的白名单/信任区中，本软件保证不含病毒，您可以亲自审查代码，如果还是觉得不放心可以不使用

Q:打开好慢

A:Python的运行效率不高，慢属于正常现象
